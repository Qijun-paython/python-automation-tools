# Python 自动化脚本合集
> 用 AI 辅助开发的 Python 实用脚本，专注解决重复性工作。
## 已完成功能
### 1. 图片批量重命名脚本 (`rename_files.py`)
- **功能**：将文件夹内所有 `.jpg` 图片按 `日期+序号` 格式自动重命名
- **场景**：摄影师整理照片、电商卖家管理商品图
- **技术点**：os 文件操作、datetime 日期处理、正则匹配
## 使用说明
1. 确保已安装 Python 3.10+
2. 将脚本放入目标文件夹，运行 `python rename_files.py`
3. 所有 `.jpg` 文件将自动重命名为 `20250523_001.jpg` 格式
